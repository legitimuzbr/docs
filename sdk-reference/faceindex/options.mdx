<ResponseField
  name="apiURL"
  type="string"
  required
>
  É o endereço da Legitimuz API.

  Exemplo: `https://api.legitimuz.com`
</ResponseField>

<ResponseField
  name="appURL"
  type="string"
  required
>
  É o endereço do Legitimuz Verify.

  Exemplo: `https://liveness.legitimuz.com`
</ResponseField>

<ResponseField
  name="token"
  type="string"
  required
>
  O token é o seu token de integração. Você pode gerar um token de integração na
  página [integrações](https://dashboard.legitimuz.com/dashboard/integrations)
  na aba de `instalação` do Dashboard Legitimuz.
</ResponseField>

<ResponseField
  name="lang"
  type="string: pt | en | es"
  default="pt"
>
  A opção lang é o idioma que será utilizado no fluxo de [Face Index](./introduction).
</ResponseField>


<ResponseField
  name="onSuccess"
  type="function: (eventName: 'faceindex'): void"
>
  O callback de sucesso é chamado quando ocorre `sucesso` em alguma etapa do fluxo
  de [Face Index](./introduction).
  Como parâmetro, é passado uma [string](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript/Reference/Global_Objects/String)
  com a etapa que ocorreu sucesso.

  Exemplos:

  <CodeGroup>
    ```html HTML
    <div id="legitimuz-root"></div>
    <script src="https://cdn.legitimuz.com/js/sdk/faceindex.js"></script>
    <script type="text/javascript">
      const sdkInstance = LegitimuzFaceIndex({
        /* opções */
        onSuccess(eventName) {
          console.log(eventName); // faceindex
        },
      });

      sdkInstance.mount();
    </script>
    ```

    ```jsx JSX
    import { useRef } from 'react';

    function Component() {
      const sdkInstance = useRef(null);

      const handleOnLoadSDK = () => {
        sdkInstance.current = LegitimuzFaceIndex({
          /* options */
          onSuccess(eventName) {
            console.log(eventName); // faceindex
          },
        });

        sdkInstance.current.mount();
      }

      return (
        <>
          <input id="legitimuz-hydrate-cpf" type="tel" />
          <script
            src="https://cdn.legitimuz.com/js/sdk/faceindex.js"
            onLoad={handleOnLoadSDK}
          ></script>
        </>
      );
    }
    ```

  </CodeGroup>
</ResponseField>
