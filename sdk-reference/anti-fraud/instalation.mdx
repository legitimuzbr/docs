### Exemplo

Para instalar o Anti Fraude SDK em seu projeto, copie e cole o código abaixo na página que deseja utilizar o SDK.

```html HTML
<script src="https://cdn.legitimuz.com/js/sdk/antifraude.js"></script>
<script type="text/javascript">
  const sdkInstance = LegitimuzAntiFraude({
    apiURL: "https://api.legitimuz.com",
    token: "seu-token-de-integração",
    action: "signin",
  });

  // Montar o SDK
  sdkInstance.mount();
</script>
```

### Uso

Antes de implementar o código a seguir, é importante garantir que a página
contenha os elementos HTML necessários.
O Face Index SDK requer um elemento de entrada [input](https://developer.mozilla.org/pt-BR/docs/Web/HTML/Element/Input)
com o atributo [id](https://developer.mozilla.org/pt-BR/docs/Web/HTML/Global_attributes/id) definido
como [legitimuz-action-send-analisys](./actions).
Este elemento é usado para a entrada do CPF.

```html
<input id="legitimuz-hydrate-cpf" type="tel" max-length="11" />
```

Além disso, a página deve incluir um botão [button](https://developer.mozilla.org/pt-BR/docs/Web/HTML/Element/button)
com o atributo id definido como [legitimuz-action-verify](./actions).
Esse botão é essencial para enviar os dados para análise.

<Warning>
  Certifique-se de que esses elementos estejam presentes na página antes de
  prosseguir com a implementação do código acima.
</Warning>

<Tip>
  Se seu código possue um padrão de escrita diferente? fique tranquilo, você
  pode alterar o seletor dos elementos através dos [métodos](./actions)
  exportados pelo SDK.
</Tip>
