<ResponseField
  name="host"
  type="string"
  required
>
  O host é o endereço da API Legitimuz.

  Exemplo: `https://api.legitimuz.com`
</ResponseField>

<ResponseField
  name="token"
  type="string"
  required
>
  O token é o seu token de integração. Você pode gerar um token de integração na
  página [integrações](https://dashboard.legitimuz.com/dashboard/integrations)
  na aba de `instalação` do Dashboard Legitimuz.
</ResponseField>

<ResponseField
  name="lang"
  type="string: pt | en | es"
  default="pt"
>
  A opção lang é o idioma que será utilizado no fluxo de [validação OCR + Facematch](/sdk-reference/legitimuz/ocr-facematch).
</ResponseField>

<ResponseField
  name="enableRedirect"
  type="boolean"
  default="false"
>
  A opção enableRedirect é um booleano que define se após o fluxo de
  [validação OCR + Facematch](/sdk-reference/legitimuz/ocr-facematch)
  o usuário será redirecionado para o link cadastrado no `Dashboard Legitimuz`.
</ResponseField>

<ResponseField
  name="autoOpenValidation"
  type="boolean"
  default="false"
>
  A opção autoOpenValidation é um booleano que define se o `popup` de
  de [validação OCR + Facematch](/sdk-reference/legitimuz/ocr-facematch) será aberto automaticamente
  após a entrada de valor no [input](https://developer.mozilla.org/pt-BR/docs/Web/HTML/Element/Input)
  de [cpf](/sdk-reference/legitimuz/fields).
</ResponseField>

<ResponseField
  name="onSuccess"
  type="function: (eventName: 'ocr' | 'facematch'): void"
>
  O callback de sucesso é chamado quando ocorre `sucesso` em alguma etapa do fluxo
  de [validação OCR + Facematch](/sdk-reference/legitimuz/ocr-facematch).
  Como parâmetro, é passado uma [string](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript/Reference/Global_Objects/String)
  com a etapa que ocorreu sucesso.

  Exemplos:

  <CodeGroup>
    ```html HTML
    <div id="legitimuz-root"></div>
    <script src="https://cdn.legitimuz.com/js/sdk/legitimuz-sdk.js"></script>
    <script type="text/javascript">
      const sdkInstance = Legitimuz({
        /* opções */
        onSuccess(eventName) {
          console.log(eventName); // ocr | facematch
        },
      });

      sdkInstance.mount();
    </script>
    ```

    ```jsx JSX
    import { useRef } from 'react';

    function Component() {
      const sdkInstance = useRef(null);

      const handleOnLoadSDK = () => {
        sdkInstance.current = Legitimuz({
          /* options */
          onSuccess(eventName) {
            console.log(eventName); // ocr | facematch
          },
        });

        sdkInstance.current.mount();
      }

      return (
        <>
          <input id="legitimuz-hydrate-cpf" type="tel" />
          <script
            src="https://cdn.legitimuz.com/js/sdk/legitimuz-sdk.js"
            onReady={handleOnLoadSDK}
          ></script>
        </>
      );
    }
    ```

  </CodeGroup>
</ResponseField>
